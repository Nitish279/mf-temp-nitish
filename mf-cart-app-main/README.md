# mf-cart-app

Ecommerce Cart Application built using Webpack 5 Module Federation. And these Micro Frontends apps are divided into three React applications.
1. Container (core)
2. Micro Frontend 1 (product)
3. Micro Frontend 2 (payment)

## Tech used:
1. React
2. Webpack
3. Redux Toolkit
4. React Router DOM
5. Module Federation
